# Lightning Strike Analysis (2016–2018)

A compact data analysis project using NOAA lightning strike data from 2016–2018.  
The notebook calculates weekly and quarterly lightning strike totals and visualizes the results using bar and line charts.

## What This Project Does
- Loads and cleans NOAA lightning data
- Creates week and quarter columns
- Aggregates total strikes by week and by quarter
- Plots:
  - Weekly bar chart
  - Quarterly trend line (in millions)

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Summary
The analysis highlights seasonal lightning patterns, showing quarterly spikes and week-level variations across 2016–2018.
